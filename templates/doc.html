Rotas da API

Essa API recupera dados dos animes de https://meusanimes.com
Cada página atualmente contém 18 animes

Method Endpoint Descrição Parâmetros Retorno
GET /episodios/lancamento Recupera os episódios mais recentes lançados Null

{"image":"Link da imagem",

GET /animes/{page}
/animes/{page}/{pages}		 Recupera todos animes, divididos por página {page}: Página atual (int)
{pages}: Páginas para serem mostradas de uma vez só (int)

{"eps":"Total de episódios disponíveis",

"name":"Nome do anime"}
 ],
"message":"18 animes carregados!"}

200 OK

/animes/{busca}/{page}/{pages}		 Recupera todos animes que satisfazem a busca, divididos por página {busca}: Nome ou parte do nome de um anime para a busca (string)
{page}: Página atual (int)
{pages}: Páginas para serem mostradas de uma vez só (int)
